# Advanced E-Commerce Sales Analysis

## Project Overview

This project analyzes Superstore sales data to identify key business insights, profitability drivers, and loss-making areas using Python.

---

## Project Objective

The objective of this project is to:

- Analyze sales and profit performance
- Identify top-performing categories
- Find loss-making products
- Analyze regional performance
- Study discount impact on profitability
- Generate business recommendations

---

## Dataset Information

Dataset: Sample Superstore Dataset

Total Records: 9,994

Columns Included:

- Ship Mode
- Segment
- Country
- City
- State
- Region
- Category
- Sub-Category
- Sales
- Quantity
- Discount
- Profit

## Key Findings

1. Technology category generated the highest sales.

2. Technology category generated the highest profit.

3. West region generated the highest profit.

4. Tables sub-category generated the highest loss.

5. Higher discounts negatively affected profitability.

6. Consumer/Top-performing segment contributed the highest sales.

---

## Business Recommendations

1. Increase investment in Technology products.

2. Review discount strategy to improve profitability.

3. Optimize pricing and cost structure for Tables.

4. Focus expansion efforts on the West region.

5. Create loyalty programs for high-value customers.
